System Phases

The system consists of several phases, each of which is designed to facilitate the CVM assessment process.

First Phase
Preprocessing Phase

In this phase, you can load the images and masks from the data folder. The data folder should contain both images and masks within the same directory. Each folder (images and masks) must include subfolders for the classes (e.g., Stage 1, Stage 2, Stage 3), and the corresponding files must have matching names. The images should be placed in the Original folder, while the masks should be in the Masks folder.

Second Phase
Features Extraction Phase

In this phase, you can extract the features from the images and masks. The extracted features will be saved on the server and can be used for further analysis.

Third Phase
Classification Phase

In this phase, you can perform the classification of the extracted features. The classification will be based on the extracted features and the selected model.

Inference Phase

In this phase, you can perform the inference phase. In it, you can upload an image and the corresponding mask to classify the image based on the extracted features and the selected model.
